# Honours Project 2025-2026
## Note about the flowcharts created
The flowcharts created (in StarUML) are available in three different formats:

- As .png files, available to view anywhere. Keep in mind that because these were created with an unregistered version of StarUML, the background will be filled with the word "unregistered".
- As an "mdj" file, viewable in StarUML.
- As a "web-app", generated by StarUML.

To view it in the "web-app" format, download the folder labelled "html-docs", and open the "index.html" file, from there, you can view the flowcharts, as well as an index containing al decision trees, individual flows, etc.
