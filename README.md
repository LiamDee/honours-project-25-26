# Instructions on how to run the Android application

## To install the application, follow these steps:
1.	Download the .apk file made by building the .apk file in Android Studio.
2.	Navigate to where the file was built.
3.	Download the application to your device
4.	Install the app by double tapping the .apk file.
## If this did not work, follow these next steps: <br>
4a. When prompted, go to the settings app and provide permission to “Install unknown apps”: If you are prompted with more instructions, follow those instructions. After this, attempt to reinstall the app <br><br>
4b. If step 4a didn’t work, enable developer mode by going to Settings --> About Phone, and tapping the Build Number (sometimes labelled Version Number) 5-7 times until a pop up appears. The specifics may vary depending on the phone model and Android version, look up instructions for your phone and/or Android version for further help. Afterwards, try reinstalling the app again. <br>
## These instructions are just a general overview of how to install the app, specific parts may be different depending on your phone, please contact myself via Email if you require assistance installing the app.
